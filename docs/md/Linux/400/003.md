# grep 搜索关键词并提取文件名

说明：

- -l 或 --files-with-matches 选项：只显示包含匹配项的文件名
- -r 或 -R 选项：递归搜索子目录
- * 表示当前目录所有文件
- . 表示当前目录及其所有子目录

示例：

```shell
# 搜索当前目录中包含"error"的文件
grep -l "error" *

# 递归搜索整个项目目录中包含"TODO"的文件
grep -rl "TODO" /path/to/project/

# 结合正则表达式搜索
grep -rl "^import" src/
```
